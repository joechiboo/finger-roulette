# 手指抽籤 Finger Roulette

一個有趣的多人手指抽籤遊戲，適合朋友聚會時決定誰要去買飲料、誰先開始等場合！

## 功能特色

- 🎯 **多點觸控** - 支援多人同時把手指放在螢幕上
- 🎨 **炫彩效果** - 每根手指都有獨特的顏色與粒子特效
- ⏱️ **可調倒數** - 1~5 秒倒數時間可自由設定
- ✨ **華麗動畫** - 旋轉掃描光束、爆炸粒子、贏家皇冠
- 📱 **震動回饋** - 支援手機震動效果
- 🔄 **自動重置** - 所有手指離開後自動重新開始

## 使用方式

1. 用手機或平板開啟網頁
2. 多人各放一隻手指在螢幕上
3. 停留指定秒數後自動開始抽籤
4. 隨機閃爍後選出幸運兒！
